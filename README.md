
🏥 Hospital Emergency Room Dashboard (Excel)

This project showcases an interactive Emergency Room (ER) Dashboard built in Excel, designed to transform raw healthcare data into actionable insights for better decision-making.


📊 Project Overview

The dashboard analyzes patient data and operational metrics to help healthcare professionals monitor performance, reduce wait times, and improve patient experience.

🚀 Key Features
📌 Real-time KPI Monitoring
Tracked 485+ patient records to provide a comprehensive overview of ER performance.
⏱️ Average Wait Time Analysis
Identified and monitored average wait time (35.51 minutes) to reduce decision latency.
⭐ Patient Satisfaction Insights
Integrated satisfaction score (4.71) to evaluate service quality.
🏥 Admission Trends
Visualized Admitted vs Not Admitted patients for better operational clarity.
👥 Patient Segmentation
Analyzed patient flow based on:
Age groups
Department referrals
Gender distribution
📅 Attendance Tracking
Included attendance status for deeper analysis of patient patterns.
🛠️ Tools & Technologies
Microsoft Excel (Data Cleaning & Preparation)
📈 Key Learnings
Data storytelling through interactive dashboards
KPI design for real-world business problems
Transforming raw data into actionable insights
Healthcare data analysis and visualization
🎯 Objective

To demonstrate the ability to:

Build business-focused dashboards
Extract meaningful insights from data
Support data-driven decision-making
📌 Conclusion

This project reflects my growing interest in Business Intelligence, Data Analysis, and Dashboard Development, with a focus on turning data into impactful decisions.
